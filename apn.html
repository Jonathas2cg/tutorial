Criando uma Nova APN

Seu aplicativo não conecta ou não gera dados? Siga os passos abaixo para configurar uma nova APN de forma simples.

O que é uma APN? É a configuração de acesso à rede de dados do seu dispositivo. Ajustá-la pode corrigir problemas de conexão.

Passos para criar uma nova APN:

  1. Acesse as configurações do seu dispositivo.
  2. Vá para Redes ou Conexões.
  3. Selecione Redes Móveis ou Nomes de Pontos de Acesso (APN).
  4. Toque em Adicionar (+) para criar uma nova APN.
  5. Preencha os campos com as seguintes informações:

Configurações da APN da Vivo:

Nome: MVTUNNEL
APN: smart.m2m.vivo.com.br
Proxy: Não definido
Porta: Não definido
Nome de usuário: vivo
Senha: vivo
Tipo de autenticação: CHAP
Protocolo APN: IPv4
Protocolo de roaming de APN: IPv4

Salve as configurações e selecione esta APN como padrão. Sua conexão estará pronta para uso!
